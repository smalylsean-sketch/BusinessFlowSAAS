
## Endpoints

### Authentication
- `POST /api/auth/signup` - Create a new account
- `POST /api/auth/login` - Login to existing account

### Dashboard
- `GET /api/dashboard` - Get dashboard overview data

### Invoices
- `GET /api/invoices` - Get all invoices for current company
- `POST /api/invoices` - Create a new invoice
- `GET /api/invoices/:id` - Get specific invoice
- `PUT /api/invoices/:id` - Update invoice
- `DELETE /api/invoices/:id` - Delete invoice

### Payroll
- `GET /api/payroll` - Get payroll data
- `POST /api/payroll` - Process payroll

### Reports
- `GET /api/reports/pl` - Profit & Loss report
- `GET /api/reports/balance-sheet` - Balance sheet report
- `GET /api/reports/vat` - VAT report

### Wallets
- `GET /api/wallets` - Get all wallets
- `POST /api/wallets/transfer` - Transfer funds between wallets
